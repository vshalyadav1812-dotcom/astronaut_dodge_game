# Astronaut Dodge

A fast-paced, space-themed dodge game built with HTML, CSS, and JavaScript. Navigate your astronaut through the solar system, avoiding asteroids, alien ships, and collecting power-ups!

## Features

- **Solar System Tour**: Journey from Mercury to Pluto and beyond. The background planet dynamically changes as your score increases!
- **Entities**: 
  - ☄️ **Asteroids**: Dodge these rotating rocks.
  - 👽 **Aliens**: Fast-moving extraterrestrials.
  - 🚀 **Ships**: Other vessels passing by.
  - 🥤 **Energy Drinks**: Collect these for an extra life!
  - 📜 **Secret Scrolls**: Grab these to pause the game and learn a random space fact!
- **Dynamic Difficulty**: The game progressively speeds up as your score climbs.
- **Web Audio API**: Procedurally generated sound effects for hits, power-ups, and secrets without needing external audio files.
- **Cross-Platform**: Play with Left/Right Arrow keys on desktop, or touch controls on mobile devices.

## How to Play

1. Open `index.html` in your web browser.
2. Click "Start Mission".
3. Use the **Left/Right Arrow Keys** or **Swipe Left/Right** on touch devices to move your astronaut.
4. Avoid incoming obstacles to stay alive.
5. Collect Energy Drinks to restore lives.
6. Collect Secret Scrolls to uncover space facts.

## Technologies Used

- **HTML5**: Structure and DOM elements.
- **CSS3**: Styling, layout, animations, and inline SVG Data URIs for all graphics (no external images required!).
- **Vanilla JavaScript**: Game loop, logic, collision detection, and Web Audio generation.
